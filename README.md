# Afago em Fio

Projeto React + TypeScript + Tailwind CSS + shadcn/ui criado a partir do HTML `croche-store.html`.

## Rodar localmente

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run build
npm run lint
```

## Estrutura

- `src/components/layout`: navbar, footer e cursor customizado.
- `src/components/sections`: componentes de cada secao da pagina.
- `src/components/ui`: componentes base no estilo shadcn.
- `src/components/illustrations`: SVGs reutilizaveis da identidade visual.
- `src/data`: conteudo de produtos, menu, marquee e passos.
